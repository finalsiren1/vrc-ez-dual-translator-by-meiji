# ショートカットキー (ホットキー)

SyncVRCは**グローバルホットキー**をサポートしています。つまり、VRChatからAlt-Tabで抜け出したり、アプリケーションウィンドウを最前面に表示したりすることなく、翻訳システムを制御できます。

これらのホットキーは、**設定(Settings)**タブの下部で設定できます。マウスでボタンを手動でクリックしたい場合は、これらを`None`（なし）に設定できます。

## 利用可能なホットキーアクション

### 1. 自分の音声翻訳の開始 (Start Outgoing) (デフォルト: F2)
* **機能:** 自分の音声翻訳(Outgoing)システムのオンとオフを切り替えます。
* **使用方法:** キーを1回押すとマイクの聞き取りを開始します。もう一度押すと停止します。

### 2. 相手の音声翻訳の開始 (Start Incoming) (デフォルト: F3)
* **機能:** 相手の音声翻訳(Incoming)システムのオンとオフを切り替えます。
* **使用方法:** キーを1回押すと他のプレイヤーの聞き取りを開始します。もう一度押すと停止します。

### 3. ホールドして話す / プッシュモード (Hold to Speak / Push Mode) (デフォルト: None)
* **機能:** 翻訳のためのプッシュ・トゥ・トーク(Push-to-Talk)ボタンとして機能します。
* **重要な要件:** このホットキーは、翻訳(Translation)タブで**Push to Record**モードを選択しており、かつ自分の音声翻訳(Outgoing)システムがアクティブに実行されている場合に**のみ機能**します。
* **使用方法:** 話している間、キーを**押し続け**ます。話し終えたらキーを離すと、即座に音声が処理され、翻訳がVRChatに送信されます。

## サポートされているキー

標準的なVRChatの操作（移動用の`WASD`やゲーム内マイク用の`V`など）との競合を防ぐため、現在SyncVRCはホットキーを以下の安全なオプションに制限しています：

* `F1` から `F12` まで
* `Insert`、`Delete`、`Home`、`End`、`PageUp`、`PageDown`、`Pause`
