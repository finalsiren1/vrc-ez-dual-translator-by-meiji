# 단축키 (핫키)

SyncVRC는 **전역 단축키(Global Hotkeys)**를 지원합니다. 즉, Alt-Tab으로 VRChat을 벗어나거나 애플리케이션 창을 맨 앞으로 가져올 필요 없이 번역 시스템을 제어할 수 있습니다.

이러한 단축키는 **설정(Settings)** 탭의 하단에서 구성할 수 있습니다. 마우스로 버튼을 직접 클릭하는 것을 선호하는 경우 단축키를 `None`(없음)으로 설정할 수 있습니다.

## 사용 가능한 단축키 작업

### 1. 내 음성 번역 시작 (Start Outgoing) (기본값: F2)
* **기능:** 내 음성 번역(Outgoing) 시스템을 켜고 끕니다.
* **사용법:** 키를 한 번 누르면 마이크 듣기가 시작됩니다. 다시 누르면 중지됩니다.

### 2. 상대방 음성 번역 시작 (Start Incoming) (기본값: F3)
* **기능:** 상대방 음성 번역(Incoming) 시스템을 켜고 끕니다.
* **사용법:** 키를 한 번 누르면 다른 플레이어의 음성 듣기가 시작됩니다. 다시 누르면 중지됩니다.

### 3. 길게 눌러 말하기 / 푸시 모드 (Hold to Speak / Push Mode) (기본값: None)
* **기능:** 번역을 위한 Push-to-Talk 버튼 역할을 합니다.
* **중요 요구 사항:** 이 단축키는 번역(Translation) 탭에서 **Push to Record** 모드를 선택했고 내 음성 번역(Outgoing) 시스템이 활발하게 실행 중인 경우에**만 작동**합니다.
* **사용법:** 말하는 동안 키를 **길게 누릅니다**. 문장을 끝마치고 키에서 손을 떼면 즉시 오디오가 처리되고 번역이 VRChat으로 전송됩니다.

## 지원되는 키

표준 VRChat 컨트롤(이동을 위한 `WASD` 또는 게임 내 마이크를 위한 `V` 등)과의 충돌을 방지하기 위해, SyncVRC는 현재 단축키를 다음의 안전한 옵션으로 제한합니다.

* `F1` 부터 `F12` 까지
* `Insert`, `Delete`, `Home`, `End`, `PageUp`, `PageDown`, `Pause`
