# 快捷键 (Hotkeys)

SyncVRC 支持**全局快捷键**，这意味着您无需通过 Alt-Tab 切换出 VRChat 或将应用程序窗口调至前台即可控制翻译系统。

您可以在**设置 (Settings)** 选项卡底部配置这些快捷键。如果您更喜欢使用鼠标手动点击按钮，则可以将其中任何一个设置为 `None`（无）。

## 可用的快捷键操作

### 1. 开启输出翻译 (Start Outgoing)（默认：F2）
* **功能：** 开启或关闭输出 (Outgoing) 翻译系统。
* **用法：** 按一次该键开始监听您的麦克风。再按一次停止。

### 2. 开启输入翻译 (Start Incoming)（默认：F3）
* **功能：** 开启或关闭输入 (Incoming) 翻译系统。
* **用法：** 按一次该键开始监听其他玩家。再按一次停止。

### 3. 按住说话 / 按键模式 (Hold to Speak / Push Mode)（默认：None）
* **功能：** 充当翻译的按键发言 (Push-to-Talk) 按钮。
* **重要要求：** 仅当您在“翻译 (Translation)”选项卡中选择了 **Push to Record (按键录音)** 模式且输出系统处于运行状态时，此快捷键**才会生效**。
* **用法：** 在说话时**按住**该键。说完一句话后松开该键，系统将立即处理音频并将翻译发送到 VRChat。

## 支持的按键

为了防止与标准的 VRChat 控件（例如用于移动的 `WASD` 或用于游戏内麦克风的 `V`）发生冲突，SyncVRC 目前将快捷键限制为以下安全选项：

* `F1` 到 `F12`
* `Insert`、`Delete`、`Home`、`End`、`PageUp`、`PageDown`、`Pause`
